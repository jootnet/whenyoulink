退出客户端 确定要退出轻远程客户端吗？

确定退出后，将无法远程控制此计算机

确定 取消 import { $, on } from "@sciter"; const window = Window.this; const closeBtn = $('#closeBtn') const okBtn = $('#okBtn') const cancelBtn = $('#cancelBtn') closeBtn.on('mouseenter', () => { closeBtn.firstElementChild.classList.add('hover') }) closeBtn.on('mouseleave', () => { closeBtn.firstElementChild.classList.remove('hover') }) on('click', '#closeBtn', () => { window.close(false) }) cancelBtn.on('mouseenter', () => { cancelBtn.style['background'] = '#0078D7' cancelBtn.style['color'], '#fff' }) cancelBtn.on('mouseleave', () => { cancelBtn.style['background'] = 'transparent' cancelBtn.style['color'] = '#0078D7' }) on('click', '#okBtn', () => { window.close(true) }) on('click', '#cancelBtn', () => { window.close(false) })
